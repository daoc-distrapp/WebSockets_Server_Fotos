# WebSockets_Server_Fotos
Servidor WebSockets

### Notas
- Este es un proyecto Spring Boot

### Ejecución

- Alternativa 1
  - En la línea de comando vaya al directorio raíz (aquel con el pom.xml) y ejecute:
    - mvnw spring-boot:run

- Alternativa 2
  - En la línea de comando vaya al directorio raíz (aquel con el pom.xml) y ejecute:
    - mvn clean package
    - cd target
    - java -jar fotos_ws-0.0.1-SNAPSHOT.jar
